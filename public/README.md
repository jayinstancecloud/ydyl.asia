# Public Assets

## Logo Image

Place your coin logo image here as `logo.png`.

**Requirements:**
- File name: `logo.png`
- Recommended size: 512x512 pixels or larger (square)
- Format: PNG with transparent background preferred
- The image will be automatically used as:
  - Website logo (in navigation and footer)
  - Favicon (browser tab icon)

The logo should be the coin image with the eagle and "THE CRAZY ONES" text.
